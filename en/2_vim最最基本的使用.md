###vim最最基本的使用

####五星级常用命令 ★★★★★

#####模式切换 ★★★★★

    esc     # 普通模式
    i       # 插入模式
    v       # 视图模式
	R       # 替换模式
    :       # 命令模式
    /       # 查找模式

><b>Tips:关于模式切换</b><br>
	模式切换是理解及学习vim的起步，很多人因为不理解vim的模式而觉得麻烦而放弃使，vim默认打开的模式为普通模式，不能输入，需要按i后才能输入。是其实正是因为vim有多种模式才使得vim强大与独特，所以入门之前一定要理解vim的模式。


#####普通模式常用 ★★★★★

    k       # 向上移动一行
    j       # 向下移动一行
    h       # 向左移动一格
    l       # 向右移动一格
    u       # 撤销操作
    yy      # 复制当前行
    dd      # 删除当前行
	w       # 移动一个单词
	ctrl+d  # 向下滚动半屏
	ctrl+u  # 向上滚动半屏
	gg      # 跳到行首
	G       # 跳到最后一行
	x       # 删除一个字符并留在普通模式
	o       # 插入一行，并进入插入模式

#####命令模式常用 ★★★★★

    :wq     # 保存退出11
    :w      # 保存但不退出
    :2,10y  # 复制2到10 行
	:%s/hello/helloworld/g  # 将全文hello替换为helloworld

　
####四星级常用命令★★★★☆

#####普通模式常用 ★★★★☆

    w       # 向后移动1个单词到词首
    b       # 向前移动1个单词到词首
    gj      # 向下移动一个位置（长行非常有用)
    gk      # 向上移动一个位置（长行非常有用)
	ctrl+e  # 向上翻一行
	ctrl+y  # 向下翻一行
	$       # 光标定位到行尾

####三星级常用命令★★★☆☆

#####普通模式常用 ★★★☆☆
	ggvG=   # 自动格式化代码

#####命令模式常用 ★★★☆☆

    :tabnew    # 新建标签 tips: 常用可以map
    :tabclose  # 关闭标签 
	:tabp      # 上一个标签
	:tabn      # 下一个标签
