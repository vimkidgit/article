#### 五星级推荐插件★★★★★
	MRU         # 快速打开文件历史记录
	NERDTREE    # 目录树
	Easymotion  # 当前页快速移动
	tagbar      # 显示文件类名,方法及变量与taglist类似
    vundle      # 插件自动管理工具

#### 四星级推荐插件★★★★☆
	Dict        # 英语翻译
	vim-multiple-cursors  # 鼠标多焦点
    ctags       # 标记插件

#### 三星级推荐插件★★★☆☆
	dbext       # 数据库查询
    Vim-airline # 状态栏样式
	txtBrowser  # 格式化查看txt文件
    powerline   # 状态栏样式
